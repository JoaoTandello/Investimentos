# Aula 6

## Sobre o projeto
O projeto tem como objetivo desenvolver formulários utilizando Flutter, com foco no aprendizado por meio de desafios. Os formulários propostos incluem: simulador de financiamento, investimento e uma calculadora de bitola. Além disso, é necessário criar protótipos no Figma, reproduzindo as telas desenvolvidas no Flutter e possibilitando a interação.

## Como executar:
Para executar os projetos, siga as instruções a seguir:
1. Aperte no botão azul "<> Code" e copie o link do repositório (HTTPS)
2. Em seu gerenciador de arquivos, clicle com o botão direito em alguma pasta de sua preferência e abra o terminal (ou o `git bash here`, se tiver)
3. Digite `git clone [link]`, onde no lugar de `[link]`, você deve colocar o link do repositório que copiou anteriormente
4. Após clonar o repositório, entre na pasta com o nome do projeto, depois na pasta `"flutter_[nome]"`
5. Abra a pasta no VSCode
6. Já no VSCode, ache a pasta "lib", expanda ela e clique no arquivo `main.dart`
7. No canto diteito, aperte num botão chamado `run whitout debbuging`
8. Espere a página abrir
